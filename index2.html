Management Modules


Interest in: Internet Systems


Core full-year module:

- 6CCS3PRJ - Individual Project - 30 credits

Credit Value : 30
Coordinator: Dr Natalia Criado-Pacheco
Semester: 1 & 2
Assessment structure: 100% final report
Description: The project provides you with the oportunity to engage in a self-managed and detailed investigation of an aspect of computer science. You will develop skills in relation to: literature searching, retrieval and analysis , report writing, the oral presentation of material , project planning and management.

Term 1 :

Optional:

- 6CCS3PAL Parallel Algorithms - 15 Credits  

Credit Value : 15
Coordinator: Professor Colin Cooper
Semester:1
Assestment structure: 100% written examination
Description:On successful completion of this module you will understand the creation of efficient parallel algorithms in a range of application areas, including sorting, matrix and graph based problems.

- 6CCS3GRS Computer Graphics Systems - 15 Credits  

Credit Value: 15
Coordinator: Dr Richard Overill
Semester: 1
Assessment structure: 100% written examination
Description: This module aims to introduce you to the computational, mathematical and physical principles underlying computer graphics systems. Upon following this module successfully you will be able to apply the computational, mathematical and physical principles that are introduced to create realistic 2D, 3D Wireframe and 3D rendered images and animations.

- 6CCS3CIS Cryptography and Information Security - 15 Credits  

Credit Value:15
Coordinator:Professor Luca Vigano
Semester:1
Assestment structure:100% written examination
Description:On successful completion of this module you should be able to understand the relevant mathematical techniques associated with cryptography; understand the principles of cryptographic techniques and perform implementations of selected algorithms in this area; appreciate the application of security techniques in solving real-life security problems in practical systems.

- 6CCS3CFL Compilers and Formal Languages - 15 Credits  

Credit Value:15
Coordinator:Dr Christian Urban
Semester:1
Assestment structure:80% written examination, 20% coursework
Description:At the end of this module students will have learned how to use regular expressions to scrape information from the web, how to design grammars for parsing languages and how to implement a small interpreter and compiler. Students will be able to implement the central components of a small compiler. Students will also know the theory behind lexing and parsing so that they canchoose an appropriate algorithm for recognising a computer language.

Term 2 :

Optional:

- 6CCS3NSE Network Security - 15 Credits  

Credit Value: 15
Coordinator: Professor Luca Vigano, Dr Jose Such
Semester: 2
Assessment structure: 100% written examination
Description: This module introduces the underlying concepts and practical applications of information security engineering at a level that reflects the current state-of-the-art. Successful completion of this course will provide students with a detailed understanding of how different types of attacks on computer systems and networks are carried out and also how such attacks can be reliably prevented or detected.

- 6CCS3SAD Software Architecture and Design - 15 Credits  

Credit Value: 15
Coordinator: Dr Hamza Hamza
Semester: 2
Assessment structure: 100% written examination
Description: This module aims to explore the practice of software architecting as applied to the development of enterprise software systems. To learn about software architecture, architecture patterns, frameworks, design patterns, pattern languages, layers of change, the architecting process and the practical process of software design and implementation. Ideas are put into practical perspective through an introduction to the UML2 superstructure and enterprise component middleware. By the end of this module, the student should have a broad knowledge of the issues, techniques and processes involved in architecture design, and the ability to design medium to large size software systems.

- 6CCS3SIA Software Engineering of Internet Applications - 15 Credits  

Credit Value: 15
Coordinator: Professor Stefan Edelkamp
Semester: 2
Assessment structure: 100% written examination
Description: This module will cover different approaches for building internet applications and the choice of design techniques and technologies involved from a software engineering perspective, considering issues of efficiency, modularity and maintainability. You will gain the ability to design and implement internet-based systems in a modular manner, using appropriate languages and techniques.
